# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in our repository](https://github.com/changesets/changesets).

We have a quick list of common questions to get you started engaging with this project in
[our documentation](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).

## StanzaChat specifics

- All workspace packages are `private: true`; changesets is configured with `privatePackages: { version: false, tag: false }` so `changeset version` will not bump versions or create tags. Version bumps and git tags are owned by `release-please` (see `release-please-config.json` and `docs/processes/release.md`).
- Changesets in this repo exist purely as human-written "what changed for users" prose, added alongside a PR via `bun run changeset`. Maintainers may consume these into the release notes that `release-please` opens.
- Do not run `changeset publish`; the packages are not published to any registry.
